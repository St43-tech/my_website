HTML List Tags

1. Unordered List (ul)

An unordered list displays items with bullet points.

  • HTML
  • CSS
  • JavaScript
  • Python

2. Ordered List (ol)

An ordered list displays items with numbers.

  1. Wake up
  2. Brush teeth
  3. Have breakfast
  4. Go to school

3. Ordered List with Letters (ol type="A")

  1. Mango
  2. Banana
  3. Orange
  4. Pineapple

4. Ordered List with Roman Numerals (ol type="I")

  1. Introduction
  2. Main Body
  3. Conclusion

5. Nested List

A list inside another list.

  • Fruits
    • Mango
    • Banana
  • Vegetables
    • Carrot
    • Spinach
  • Grains
    1. Rice
    2. Wheat

A description list displays terms and their definitions.
